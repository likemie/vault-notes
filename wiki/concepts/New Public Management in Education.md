## 定义

新公共管理（New Public Management, NPM）在教育中的应用指过去约 30 年间，西方教育系统以新管理主义理念为指导所经历的重大改革浪潮。其核心特征是用企业管理逻辑重塑公共教育（Møller, 2017, p.381-382）。

> Møller 描述其特征为："绩效标准与测量、产出控制、供给竞争和管理问责实践"（standards and measures of performance, output control, competition in provision and managerial accountability practices）（Møller, 2017, p.381）。

## 概念辨析

- vs **专业实践共同体话语** — Møller 指出了一个悖论：OECD 同时推崇两个表面上矛盾的话语（Møller, 2017, p.377）：
  1. 将学校从科层制转变为专业实践共同体的分布式领导建议（社区隐喻）
  2. 新公共管理的外部控制话语（市场隐喻）
  - 这两个话语并非互相替代，而是**并行运作**（operate in tandem）
- → [[Performativity of Measurement]] — NPM 是绩效文化的制度根源，绩效测量是 NPM 的技术手段
- → [[Evidence-Based Education]] — "什么有效"议程在 NPM 框架下获得额外的制度驱动力

## 核心要素

Møller（2017, p.381-382）从学校领导者的日常体验出发，描述了 NPM 的具体表现：

- **合同外包与外包化**（contracting, outsourcing）
- **公共关系**（public relations）
- **基准测试**（benchmarking）
- **考试成绩**（test scores）成为工作的中心舞台
- **管理问责**（managerial accountability）取代专业自主

学校领导者的时间被以下特征所主导：不可预测性、大量不确定性、去规制化和管理问责，导致经济效率需求常常压倒集体和公共利益（Møller, 2017, p.381-382, citing Blackmore, 2011; Thomson, 2009）。

## 概念演变

NPM 在教育领域的渗透经历了从经济学理论到全球治理工具的制度化过程：

- **1950s** — 经济学理论开始将"[[Knowledge-Based Economy | 知识]]"作为生产要素讨论，知识的经济化逻辑初步形成（Amos, 2022, p.56）
- **1980s–1990s** — NPM 从公共行政领域扩展至教育系统，西方各国推动学校自主权、问责制和市场化改革（Møller, 2017, p.381）
- **2006** — OECD 发布 "Improving School Leadership" 报告，同时推动 NPM 问责话语和分布式领导话语——两个表面上矛盾的话语开始并行运作（Møller, 2017, p.377）
- **2011** — Eacott 论证 NSW 校长培养方案是"当代国家对公共部门管理主义干预的一部分"（Eacott, 2011, p.45）
- **2022** — Amos 诊断 NPM 术语已"深刻嵌入我们当前的教育结构"（Amos, 2022, p.56）
- **2022** — Rizvi 记录 NPM 在高等教育国际化中的运作：NPM"理念、技术和实践从企业部门被广泛引入，以使公立高等教育机构更像企业运作"。在澳大利亚，大学在招收全额付费国际学生方面的成功使政府有底气推动其他市场导向政策（Rizvi, 2022, pp.97–98）

## 理论基础

- [[Transitology]] — Cowen 的晚期现代教育双轴模型将 NPM 定位为市场驱动转型的制度操作工具：纵轴从 polis-driven 到 market-driven，核心价值从"教育机会平等"转为"内部效率与外部效能"（Amos, 2022, p.55）
- [[Performativity of Measurement]] — NPM 是绩效文化的制度条件，绩效测量是 NPM 的技术手段
- [[Evidence-Based Education]] — "什么有效"议程在 NPM 框架下获得制度驱动力与管理合法性

## 实证发现

基于 LE@DS（Leading Democratic Schools）研究网络的发现（Møller, 2017, p.381-382, citing Gunter et al., 2016）：

- 标准和标准化几乎已成为学校改革者的"咒语"（mantra）
- 许多国家经历了日益增长的**绩效文化**（culture of performativity）发展趋势
- NPM 改革**分散了对公平问题的注意力**（Møller, 2017, p.382, citing Hall et al., 2015）
- 管理者和学校领导者都将数据的生产和使用视为解决学校和 student 层面问题的"合法方式"（Møller, 2017, p.382, citing Sellar, 2015）
- 结果：教育劣势问题倾向于被框定为**技术问题**，而非权力与资源分配问题

- Amos (2022, p.56) 观察：NPM 的术语——效率、效能、绩效、质量保证、公私合作伙伴关系、创业教育——已"深刻嵌入我们当前的教育结构"（deeply ingrained in our current educational structures），覆盖从 K-12 到高等教育的所有层级。这一深度嵌入使得 NPM 的话语和实践被视为不言自明的前提

## 争议与批评

### 民主 vs 市场的根本张力

Møller（2017, p.382）识别出 NPM 在教育中的主要张力：

- **一方**：竞争与私有化话语，支撑新公共管理
- **另一方**：根植于社会民主意识形态的话语——公平、参与、综合教育

这种张力在"教育作为公共产品"（education as a public good）的论述中尤为显著。Møller 呼吁更多地了解维持教育作为公共产品的条件和过程（Møller, 2017, p.382）。

### 对教育[[School Leadership | 领导力]]研究的影响

在 NPM 环境中，校长日益经历一种工作环境，其中经济利益或效率需求经常压倒集体和公共利益（Møller, 2017, p.381-382）。Møller 认为仅研究组织内部的领导力行为是不够的——必须将社会视角（关注政治经济变化）与组织视角同等重视（Møller, 2017, p.381）。

### 领导力培养作为管理主义国家干预

Eacott（2011）论证标准化校长培养方案是 NPM 干预教育的关键机制：

- NSW 校长培养不过"是当代国家对公共部门管理主义干预的一部分"——通过学校本位规划、择优晋升、领导力能力框架和竞争性入学等结构安排，将校长身份建构为"国家启动的改革的交付者"（Eacott, 2011, p.45）
- 立法（NSW 立法规定校长对"有效和高效运行"负责）以经济语言表述政策声明，专业标准体现 Taylor 主义遗产（"最佳实践"）和市场意识形态（"学校形象维护"）（Eacott, 2011, p.52-54）
- 领导力培养 doxa 将"证据本位策略"和技术-理性主义框定为不言自明的前提，消除了对"谁决定什么是重要的"和"学校成功的衡量标准由谁设定"等前提性问题的追问（Eacott, 2011, p.50）
- 当国家测试制度（NAPLAN）、国家课程和 MySchool 网站全部建立后，"国家对教育的管理主义项目几乎完成"（Eacott, 2011, p.54）

## 相关案例／政策

- [[OECD Improving School Leadership Project 2006]] — OECD 同时推动了 NPM 问责话语和分布式领导话语，体现了矛盾的并行运作
- [[ISSPP Project]] — 发现在高 stakes 问责体制（如英语国家）与低 stakes 福利传统（如北欧国家）中，校长面临根本不同的政策环境
- [[School Autonomy]] — 澳大利亚学校自主权改革集中体现了 NPM 的市场逻辑（竞争、经济效率、公共问责）如何在学校治理层面被操作化
- [[My School Website]] — NPM 产出控制和公共问责的典型工具，以市场逻辑驱动学校间竞争

### 澳大利亚：学校自主权改革中的 NPM 逻辑

Keddie et al.（2020）基于对 42 位教育利益相关者的访谈，详细记录了 NPM 的市场逻辑如何通过学校自主权改革在澳大利亚公立教育系统中运作并产生不平等。

#### 经济效率与国家回撤

伴随学校自主权改革的经济理性主义"倾向于破坏传统上与支持公立学校相关的中央教育官僚机构的结构、资源和项目"（Keddie et al., 2020, p.433）。在 WA，正如 1990s 的 Victoria，这种逻辑导致了区域层面学校支持结构的**拆除**，对小规模、农村和偏远学校产生灾难性后果。类似地，在英格兰，"以学校自主权改革的名义——即 Academies 运动以及更早的自我管理学校——国家教育系统的区域支持（即地方教育当局）被削减，使学校自行管理先前由国家负责的义务"（Ball & Junemann, 2012; Smyth, 2011，引自 p.433-434）。

Edward（union, NSW）将教育部描述为权力下放政策前自身的一个"**空壳**"（mere husk）——"撤除了公平单位和区域/农村支持服务"（p.437）。Delia 和 Nancy 评论称"在自主权和权力下放的伪装下"削减资金"使学校失去了处理各种公平问题所需的关键建议或支持"（p.437）。

#### 竞争与残余化

NPM 的市场逻辑使学校被迫为生存而相互竞争。利益相关者描述了竞争如何创造了一个"吸取系统生命的等级秩序"（Miriam & Kyle, bureaucrats, NSW），导致公立系统内**残余化**和**社会隔离**的加剧——围绕语言、文化和宗教差异形成等级（Keddie et al., 2020, p.437）。

竞争的规模经济效应：有 3,000 名学生"你绝对大赚特赚"，但只有 400 名"你必须重组"（Russell, union, WA）。小规模学校面临系统性劣势——"你不得不这样做，因为这让你的家长满意，你不会让他们把孩子送走"（p.440）。

#### 家长筹资不平等

NPM 将责任下放到学校层面（包括财务责任），导致了系统性的家长筹资不平等。Wendy（bureaucrat, Vic）描述：

> "这是一个阶级制度；非常明确的阶级制度。如果东郊的一所学校举办校庆，他们轻松筹集 30,000 澳元……西/南郊的一所学校能筹集 2,000 澳元就算幸运了，并且大部分工作由教师完成。"（Keddie et al., 2020, p.441）

Rowe & Perry（2020）的研究发现：高 SES 地区的学校可以从家长那里获得高达四倍于低 SES 地区学校的收入。学校被迫使用"学校筹集的资金来支付本应由国家资助的服务和资源"（Nancy, union, Vic，引自 p.441）。讽刺的是，**20 世纪的中央官僚机构正是建立在为农村和区域学校提供更公平教育的基础上**，因为它们为整个系统的当地学校提供了一系列服务（Keddie et al., 2020, p.441）。

#### 人员配置的分层

NPM 去中心化的人员配置模式使"难以配置人员的学校（如农村/偏远学校）变得更加难以配置人员——导致经验不足的教师在这些学校中过多代表"（Gavin & McGrath-Champ, 2016，引自 Keddie et al., 2020, p.441）。这产生了一种"残余化的人员配置状况"——这些学校"不得不接受任何能招到的人"（Nathan, principal association, National，引自 p.441）。
