# vault-schema.md — Manifest Index Patch

Use this logic in `vault-schema.md`:

```markdown
## Index / Manifest

- `wiki/index.md` is optional and only serves as a lightweight entry page.
- Do not use Dataview as the AI-facing index because Dataview renders only inside Obsidian.
- `wiki/manifest.json` is the AI / Claude Code machine-readable index.
- `wiki/manifest.md` is the static Markdown index for GitHub / Quartz / human reading.
- Both files are generated by `scripts/update_manifest.py`.
- Do not manually maintain `manifest.json` or `manifest.md`.

### Entry existence check

1. Read `wiki/manifest.json` first.
2. Match candidate entries by:
   - `title`
   - `aliases`
   - `type`
   - `path`
3. If no match or ambiguous match, search the corresponding type folder.
4. After creating, moving, or substantially modifying entries, run:
   `python3 scripts/update_manifest.py`
```
